    
    1. Miniconda (or Psi4conda):

        a. Go to: https://conda.io/miniconda.html (or http://vergil.chemistry.gatech.edu/psicode-download/1.1.html)

        b. Download the version that best fits your system.

        c. Go to the directory where you downloaded the installer and execute it as:
           bash Miniconda3-latest-Linux-x86_64.sh (or bash Psi4conda-latest-py36-Linux-x86_64.sh)


    2. CrystaLattE Conda Environment:

        a. conda create -n CrystaLattE psi4-dev python=3.6 -c psi4

        b. source activate CrystaLattE

        a. Install using: conda install -c conda-forge pycifrw
